# BuzzFinder — 제휴지면 탐색기

버즈빌 ADM 팀의 내부 도구로, 제휴 매체(앱)의 광고 노출 지면 정보와 스크린샷을 빠르게 탐색할 수 있는 웹 프로토타입입니다.

**배포 URL:** https://jettycho.github.io/Mockups/buzzfinder/buzzfinder_prototype.html

---

## 배경 및 목적

버즈빌은 100여 개의 제휴 매체 앱에 광고를 노출합니다. 영업·운영 담당자가 특정 앱의 광고 진입 경로와 실제 지면 화면을 빠르게 확인할 수 있도록, 매체 정보와 스크린샷을 한 곳에서 탐색하는 도구가 필요했습니다.

이 프로토타입은 그 도구의 UI/UX를 검증하기 위해 제작되었습니다.

---

## 주요 기능

### 탐색
- **앱 이름 검색** — 실시간 필터링 (한글/영문)
- **카테고리 필터** — 20종 카테고리 탭, 카운트 표시
- **★ 주요 매체 필터** — 10개 대표 매체만 빠르게 조회
- 검색 + 카테고리 AND 조건 동시 적용

### 정보 표시 (매체 행)
각 앱 행은 다음 순서로 정보를 표시합니다.

```
앱 이름 → 카테고리 → 진입경로 → 담당자
```

우측 영역에서 해당 앱의 지면 스크린샷을 가로 스크롤로 탐색합니다.

### 스크린샷 인터랙션
- **Hover** — 이미지 확대 + 체크박스 나타남
- **클릭** — 라이트박스에서 전체 크기로 보기
  - 좌우 화살표 또는 하단 썸네일로 이동
  - `ESC` 또는 배경 클릭으로 닫기
- **체크박스 선택** — 여러 장 다중 선택
- **다운로드** — 선택한 이미지 로컬 저장
